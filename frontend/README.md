# E-commerce Frontend

This is the React frontend for the e-commerce application.

## Structure:
- `/src/components` - React components
- `/src/App.jsx` - Main App component
- `/src/App.css` - Global styles
- `/public` - Static assets

## Features:
- Product catalog (Men's & Women's clothing)
- Shopping cart functionality
- User authentication (Login/Signup)
- Product details pages
- Responsive design with Tailwind CSS

## Technologies:
- React 18
- React Router
- Tailwind CSS
- Vite (build tool)

## Getting Started:
```bash
cd frontend
npm install
npm run dev
```
